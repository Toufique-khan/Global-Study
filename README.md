# 🌍 Global Study — Responsive Landing Page

A fully responsive landing page built using pure **HTML, CSS, and Media Queries** without using any frontend frameworks.

The main goal of this project was to strengthen my frontend development skills by building a modern, responsive, and interactive user interface from scratch.

## 🚀 Features
- ✅ Fully Responsive Design
- ✅ Mobile to Desktop Compatibility
- ✅ Clean & Modern UI
- ✅ Interactive CSS Animations
- ✅ Flipping Card Effect
- ✅ Smooth Transitions
- ✅ Structured Layout Design
- ✅ Pure HTML & CSS Implementation

## 🧠 What I Learned
While building this project, I improved my understanding of:
- Responsive web design
- Media queries for multiple screen sizes
- CSS animations and transitions
- UI layout optimization
- Interactive frontend effects
- Debugging responsive issues
- Creating modern landing page designs

## 🛠️ Tech Stack
- HTML
- CSS
- Media Queries

## 📌 Project Purpose
This project was created as a frontend practice project to improve my:
- UI responsiveness skills
- CSS animation skills
- Layout designing techniques
- Real-world frontend problem-solving abilities
